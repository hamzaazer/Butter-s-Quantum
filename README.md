## Configuration
The framework is designed to be self‑contained, but you can customize:

C2 Server: Change the TcpStream::connect address in main() (default: 127.0.0.1:4444).

Persistence: Uncomment the scheduling code (not included in the snippet).

Encryption: Modify the generate_key() function to use a different entropy source.

Payload: Replace ENCRYPTED_PAYLOAD with your own encrypted data (must be encrypted with the same key).
